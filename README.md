# KNN Classification Project

## Overview
This project uses the K-Nearest Neighbors (KNN) algorithm to perform classification on a Kaggle dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook (VS Code)

## Workflow
1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. KNN Model Training
5. Model Evaluation

## Results
- Accuracy: 94%
